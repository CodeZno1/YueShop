# YueShop
C#.net移动云商城

开发和运行环境：
开发工具：vs2015
数据库软件：SqlServer2008R2
运行框架：.netframework4.5.1
在项目运行或部署之前确保开发工具安装好。如果安装好了vs的话。.netframework4.5.1是不需要再安装，如果是服务器部署不安装vs的话，需要单独安装的。作为开发者应该都知道，如果是小白，请一定记住。
